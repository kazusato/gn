# GN Tool

# Usage

## Config file
Put your configuration file as ~/.gn/config.
The contents should be in the [TOML](https://github.com/toml-lang/toml) format
and contain the following entries:

```
Url = "https://gn-server-url/"
UserId = "YourUserId"
Password = "YourPassword"
```

## Execution
Call "gn" command.
